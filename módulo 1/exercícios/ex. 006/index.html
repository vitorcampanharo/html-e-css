Textos em negrito

Para escrever em negrito como essas palavras que escrevi agora deve-se usar a tag <strong>

Textos em itálico

Para escrever em itálico como estas palávras escritas agora deve-se usar a tag <em>

Textos marcados

É possível marcar um texto, como se fosse uma caneta marca texto, usando a tag <mark>.

Textos grandes e pequenos

É possível escrever um texto grande desta forma usando a tag <big>. (OBS: Esta tag está caindo em desuso)

E também escrever um texto pequeno desta forma usando a tag <small>

Textos subescritos e sobreescritos

É possível escrever termos sobre outros como x 20+3 usando a tag <sup>.

É possível escrever textos sobre outros como H2O usando a tag <sub>.

Texto deletado

Um texto pode ser lido, mas não deve ser considerado, usando a tag <del>.

Texto inserido

Um texto pode ser marcado para representar uma conteúdo importante, usando a tag <ins>.
