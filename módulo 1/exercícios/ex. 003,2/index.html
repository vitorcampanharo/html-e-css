Como criar uma página web

Existem três ferramentas principais para desenvolvimento web, o HTML, CSS e JavaScript

O HTML serve para construir o conteúdo da página, com textos, imágens, links, etc..

O CSS serve para ilustrar o conteúdo, dar vida com cores, edição de tamanhos, etc..

O JavaScript por sua vez, sendo uma linguagem de programação, serve para criar interações na página, para torná-la funcional, e não apenas visual.
